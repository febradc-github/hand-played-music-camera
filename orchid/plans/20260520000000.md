  Implement the following in a new file `src/calibrator.py`:

  ```python
  from __future__ import annotations

  from dataclasses import dataclass
  from enum import Enum, auto
  from typing import Optional

  import pygame

  from src.hand_tracker import HandTracker


  class CalibrationState(Enum):
      IDLE = auto()
      ONE_PRESENT = auto()
      BOTH_PRESENT = auto()
      HOLDING = auto()
      DONE = auto()


  @dataclass
  class CalibrationResult:
      left_baseline_x: float
      left_baseline_y: float
      right_baseline_x: float
      right_baseline_y: float


  # Box geometry (normalized 0-1)
  LEFT_BOX_CENTER_X = 0.25
  LEFT_BOX_CENTER_Y = 0.50
  RIGHT_BOX_CENTER_X = 0.75
  RIGHT_BOX_CENTER_Y = 0.50
  BOX_RADIUS_PX = 100
  HOLD_DURATION = 3.0
  ```

  Then implement `run_calibration(tracker, renderer, window_w, window_h)`:
  - Runs a blocking mini event/processing loop.
  - Each iteration: handle pygame events (quit/Escape returns None), call `tracker.get_frame()` and `tracker.get_fingertips()`, classify left/right fingertips by hand_label, check if each is inside its box (distance from box center in pixels < BOX_RADIUS_PX).
  - State machine: IDLE (no finger in box) -> ONE_PRESENT (one finger in its box) -> BOTH_PRESENT (both in boxes) -> HOLDING (both stable, timer counting). Timer runs when BOTH_PRESENT; resets to 0 if either finger leaves. After HOLD_DURATION continuous seconds, state = DONE.
  - Returns CalibrationResult capturing each fingertip's normalized (x, y) at the moment DONE is reached. Returns None if user quits.
  - Render each frame via `renderer.draw_calibration(frame, left_present, right_present, progress)` where progress = elapsed / HOLD_DURATION (0.0-1.0) during HOLDING, 0.0 otherwise.
