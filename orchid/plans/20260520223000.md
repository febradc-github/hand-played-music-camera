## Architecture Overview
MediaPipe Hands runs in a background thread, pushing fingertip coordinates to a thread-safe shared state. PyGame main thread reads the state, renders camera feed + two radial pie menus, resolves chord from zone hits, and sends MIDI via python-rtmidi. Chord engine maps root+modifier → MIDI note numbers using interval tables.

## File Map
### Create
- `src/__init__.py` — package marker

- `src/main.py` — main loop wiring all modules, state machine for chord changes
